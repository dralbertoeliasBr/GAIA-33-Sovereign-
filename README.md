## What is GAIA-33?

GAIA is a sovereign AI system that runs entirely on your device—your iPhone, Raspberry Pi, Android phone, or personal server. It doesn't require the cloud. It doesn't track you. It doesn't sell your data.

Instead, it becomes your **cognitive prosthetic**: remembering what you say, learning from your patterns, and helping you finish what you start—all while you keep complete control.

### The Problem We Solve

- **Centralized platforms** own your data. You create → they profit → you get nothing.
- **Cloud dependency** means your AI dies when the internet fails.
- **Privacy erosion** is the cost of using "free" services.
- **Cognitive friction** makes you lose context between sessions, restart projects, repeat work.

       ↓

## Quick Start

### Requirements
- Python 3.6+
- iPhone with Pythonista (or any device with Python)
- Internet access for setup only (optional)

### Installation

```bash
# Clone or download
git clone https://github.com/dralbertoeliasBr/GAIA-33-Sovereign-AI.git
cd GAIA-33-Sovereign-AI

# Create folder
mkdir -p ~/Documents/GAIA_UTILS

# Copy scripts
cp leitura_contexto.py ~/Documents/GAIA_UTILS/
cp doca_bridge.py ~/Documents/GAIA_UTILS/

# Test
python3 ~/Documents/GAIA_UTILS/leitura_contexto.py
```

**Expected output:**
```
✅ Contexto recuperado com sucesso!
[LEITURA] 1 blocos violet + 0 mensagens vault
CONTEXTO RECUPERADO — GAIA CONTINUIDADE
```

### How It Works

1. **Leitura (Read)**: `leitura_contexto.py` recovers your previous session context
2. **Doca (Bridge)**: `doca_bridge.py` connects your local processing to your Enclave/Qwen3
3. **Violet Box**: Immutable log of everything you say (audit trail, not surveillance)
4. **Black Box**: Sensitive acts require explicit voice confirmation
5. **Continuity**: Between sessions, you never start from zero
